Lenguajes

PHP: es un lenguaje de programación usado generalmente para la creación de contenido para páginas web. PHP es un acrónimo recurrente que significa PHP

JSP: en el campo de la informática, es una tecnología para crear aplicaciones WEB. Es un desarrollo de la compañía Sun Microsystems, y se basa en scripts, que utilizan una variante del lenguaje.

ASP.net: es una tecnología de Microsoft para páginas web generadas dinámicamente, que ha comercializado como un anexo a Internet Information Server (IIS).
